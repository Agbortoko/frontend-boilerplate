Front-end Boilerplate

Using Sass and Gulp

Here's a quick set-up guide:

  • Clone Git repo
  • Install by running "npm install" on the commmand line
  • Run "gulp" on the command line to run default task without browser sync.
  • Run "gulp sync" on the command line to run task with browser Sync
